# NOIR CAFÉ & KITCHEN

A premium restaurant/cafe website inspired by high-end Webflow templates. Dark, elegant, and cinematic design.

## Features

- **Dark Elegant Aesthetic**: Near-black background (#0B0B0B) with warm off-white text
- **Cinematic Hero**: Full-screen hero with parallax effects and floating signature dish card
- **Premium Typography**: Playfair Display serif for headings, Inter for body
- **Smooth Animations**: Framer Motion with cinematic 400-700ms transitions
- **Glassmorphism**: Backdrop blur effects throughout
- **CSS Image Placeholders**: Gradient-based "photo" panels with texture
- **Reservation-Focused**: Multiple booking CTAs and forms

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Pages

- **Home**: Hero, social proof, menu highlights, story, specials, testimonials, reservation CTA
- **Menu**: Category-filtered menu with chef's recommendations
- **Gallery**: Masonry-style gallery with filters
- **Reservations**: Full reservation form with alternative booking options
- **Contact**: Contact info + private events inquiry form

## Design System

- **Colors**: 
  - Background: #0B0B0B / #0E0E0E
  - Text: #F5F2EA
  - Muted: #C9C5B8
  - Accent: #B08D57 (bronze/gold)
- **Typography**: Playfair Display (serif), Inter (sans)
- **Spacing**: py-24 minimum, md:py-32 for sections
- **Borders**: Rounded-2xl, thin white/10 borders
- **Effects**: Glassmorphism, subtle grain texture, smooth animations

## Build

```bash
npm run build
npm start
```
